## Windows Server 2022 Lab

## Overview
This repository documents my hands-on practice with Windows Server 2022 using VMware Workstation.

## Environment
- Windows Server 2022 Standard (Evaluation)
- VMware Workstation
- Windows 10/11 Client (planned)

## Implemented Features
- Installed Windows Server 2022
- Promoted server to Domain Controller
- Created new Active Directory forest
- DNS integrated with AD
- Created Organizational Units (OU)
- Created domain users and security groups

## Skills Practiced
- Active Directory user and group management
- Windows Server administration
- Basic enterprise networking concepts
